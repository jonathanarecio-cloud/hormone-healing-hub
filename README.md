# Hormone Healing Hub · Demostración integral

Demostración funcional, responsive e instalable (PWA) del ecosistema Hormone Healing Hub.

## Abrir localmente

Desde esta carpeta ejecuta `python -m http.server 4173` y abre `http://localhost:4173`.

## Incluye

- Dashboard, tareas y recordatorios diarios.
- Diario del ciclo, temperatura basal, síntomas y tendencias.
- Cyclus Reset: 12 módulos, progreso, notas y materiales.
- Restore Method: programa 1:1 de cuatro meses, citas y plan personal.
- Calendario, suplementos, documentos y resultados de laboratorio.
- Recetas, favoritos y lista de la compra.
- Masterclasses, podcast, audios y HHH Academy.
- Comunidad, grupos, eventos, retos y funciones de seguridad.
- Membresía, preferencias de privacidad y gestión de datos.
- Espacio profesional: clientas, contenidos, permisos, moderación y flujo de resúmenes asistidos por IA con aprobación humana.
- Español, inglés y neerlandés con preferencia guardada.
- Instalación PWA y funcionamiento básico sin conexión.

## Importante

Esta es una demo de producto. Los datos se guardan solamente en el dispositivo. No deben introducirse datos médicos reales. Antes de producción requiere autenticación, backend cifrado, pagos, notificaciones, almacenamiento sanitario, consentimiento, auditoría, moderación y validación legal/RGPD. La información del ciclo es educativa y no constituye diagnóstico.
