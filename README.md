# Hormone Healing Hub

Primera versión funcional de la aplicación web responsive.

## Abrir localmente

Desde esta carpeta:

```powershell
python -m http.server 4173
```

Después abre `http://localhost:4173`.

## Incluye

- Dashboard diario
- Diario del ciclo y gráfico de temperatura basal
- Cyclus Reset (curso online)
- Restore Method (programa 1:1 de cuatro meses)
- Biblioteca de recursos
- Persistencia local del registro diario
- Selector inicial con español, inglés y neerlandés
- Preferencia de idioma guardada y modificable desde la cabecera
- PWA instalable y funcionamiento básico sin conexión

Esta es una demo de producto. Antes de producción requiere autenticación, backend cifrado, gestión de permisos, consentimiento y validación legal/RGPD.
